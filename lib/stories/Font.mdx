# Font ✏️

We divide the font within three categories: Caption, Body, and Display.

## Caption

Captions are used for small, brief pieces of text that accompany images, charts, tables, or other visual elements.

It is usually used to provide context, descriptions, or explanations for visual elements without drawing too much attention away from the primary content.

Body text is used for the main content of a page or article.

It is used to convey the bulk of information in a readable and clear manner, as it constitutes the primary narrative or informational content.

Display text is used for headings, titles, or other large text that needs to stand out.

It is used to grab attention, set the tone, and provide structure by breaking content into sections or emphasizing important information.

export const TextWrapper = ({children}) => {
return <div style={{display: 'flex', justifyContent: 'space-between', alignItems: 'start', gap: 36, borderTop: '1px solid var(--palette-greyscale-300)'}}>{children}</div>
}

export const TypeWrapper = ({children}) => {
return <p style={{font: 'var(--caption-medium-regular)', color: 'var(--palette-greyscale-800)', width: 175, margin: 0, paddingTop: 10, paddingBottom: 10}}>{children}</p>
}

export const ExampleTextWrapper = ({children, fontStyle}) => {
return <p style={{font: fontStyle, flex: 1, margin: 0, paddingTop: 10, paddingBottom: 10}}>{children}</p>
}

export const TextDetails = ({fontWeight, fontSize, lineHeight}) => {
    return <div style={{display: 'flex', flexDirection: 'column', paddingTop: 10, paddingBottom: 10, width: 175}}>
    <p style={{font: 'var(--caption-medium-regular)', color: 'var(--palette-greyscale-800)', margin: 0}}>Font Family: Poppins</p>
    <p style={{font: 'var(--caption-medium-regular)', color: 'var(--palette-greyscale-800)', margin: 0}}>Font Weight: {fontWeight}</p>
